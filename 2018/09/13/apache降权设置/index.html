apache降权设置

Posted on 2018-09-13 | In 运维

网站权限配置

这里偷了个懒使用的是phpstudy集成环境。

创建apache服务(不创建没法设置启动权限)

httpd.exe -k install -n Apache2.4

安装服务之后打开控制面板的服务，设置apache登录用户为自己创建的降权用户。

然后再简单设置一下网站目录读写权限就行了。

可以看到getshell后菜刀里显示为apache权限。

这里要说一个坑，自己在配置配置过程中出现apache无法启动的情况，或者刚启动就关闭。

原因大概有如下几种情况

1、运行库没有安装

32位的VC9运行库下载：

http://www.microsoft.com/zh-CN/download/details.aspx?id=5582

64位的VC9运行库下载：

http://www.microsoft.com/zh-CN/download/details.aspx?id=15336

2、配置路径有中文

一定要注意这个情况，任何情况都不要有中文出现，在phpmyadmin中用SQL语句读文件如果文件中含有中文字符也是读不出来的！！！

3、网站路径后有/

这样是错误的，在配置网站路径时一定要注意。

4、可以在bin目录下运行httpd.exe查看错误原因进行进一步判断
